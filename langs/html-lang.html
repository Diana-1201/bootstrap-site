Краткое описание

HTML (HyperText Markup Language) — стандартный язык разметки документов во Всемирной паутине.

Большинство веб‑страниц содержат описание разметки на языке HTML.

Пример исходного кода

<!DOCTYPE html>
<html>
<head>
  <title>Пример</title>
</head>
<body>
  <h1>Привет, мир!</h1>
</body>
</html>
