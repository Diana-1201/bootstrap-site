Краткое описание

CSS (Cascading Style Sheets) — формальный язык описания внешнего вида документа, написанного с использованием языка разметки.

Чаще всего используется как средство оформления веб‑страниц в формате HTML и XHTML, но может применяться с любыми видами документов в формате XML.

  • Позволяет отделять описание логической структуры документа от описания его внешнего вида.
  • Поддерживает медиа‑зависимые стили (для печати, экранов, мобильных устройств).
  • Даёт возможность создавать адаптивный дизайн.

Пример исходного кода

.header {
  background-color: #4CAF50;
  color: white;
  padding: 20px;
  text-align: center;
}

.container {
  margin: 20px;
  padding: 15px;
  border: 1px solid #ddd;
}
